📋 تنسيق الملفات المطلوب:

💰 ملف المدفوعات:
  • student_name (اسم الطالبة)
  • amount (المبلغ)
  • payment_date (تاريخ الدفع)
  • payment_type (نوع الدفعة - اختياري)
  • payment_method (طريقة الدفع - اختياري)
👥 ملف الطالبات:
  • name (الاسم)
  • phone (الجوال)
  • national_id (رقم الهوية - اختياري)
  • university (الجامعة - اختياري)
  • rent_amount (الإيجار - اختياري)
💸 ملف المصروفات:
  • description (الوصف)
  • amount (المبلغ)
  • expense_date (تاريخ المصروف)
  • category (الفئة - اختياري)
  • receipt_number (رقم الإيصال - اختياري)
